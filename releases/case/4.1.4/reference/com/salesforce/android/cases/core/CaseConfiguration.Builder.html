public CaseConfiguration.Builder caseListName (String caseListName)

Sets the name of the Case List. This field is required for authenticated case access.

The Case List name is the name assigned to a case list view in the Salesforce org. This value can be found by selecting edit view from the Cases tab and noting the "View Unique Name".

Parameters
caseListName The name assigned to a case list view in the Salesforce org.

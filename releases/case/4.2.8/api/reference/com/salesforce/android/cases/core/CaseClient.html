public abstract Async<CompleteCaseFeed> getCompleteCaseFeed (CompleteCaseFeedRequest request)

Retrieves all values required to display the feed on a case. Bundles the result of getCommunities(CommunitiesListRequest), getCaseDetail(CaseDetailRequest), getDefaultValues(DefaultValuesRequest), getCreateCaseLayoutData(CreateCaseQuickActionRequest) and getCaseFeed(CaseFeedRequest) into a single async.

Result handlers are fired once all calls have returned a value. If returnIntermediateResults is used in the request this could result in multiple onResult calls.

Parameters
request The CompleteCaseFeedRequest that specifies how to cache the data and which case to retrieve.
Returns
  • An Async instance which provides the CompleteCaseFeed

public Boolean isSatisfied ()

A field is satisfied if

It is not a required field and it's length is less than the max length if set. It is a required field and it's value is non zero, and less than the max lenght if set.

Otherwise if a field is not required, and has no max length is is always satisfied.

Returns
  • Whether this field has been satisfied.

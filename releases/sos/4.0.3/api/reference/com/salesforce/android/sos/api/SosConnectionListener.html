public interface

SosConnectionListener

com.salesforce.android.sos.api.SosConnectionListener

Class Overview

Interface for listening to changes in the SOS connection strength.

You can use this interface to listen to changes in the connection strength during an SOS session.

Usage Example

 

   SosConnectionListener myListener = new SosConnectionListener {
    @Override
    public void onConnectionStrengthChange (@Strength int connectionStrength) {
      // React to the change in connection strength here.
    }
   };

   Sos.addConnectionListener(myListener);

   // ... sometime later
   Sos.removeConnectionListener(myListener);

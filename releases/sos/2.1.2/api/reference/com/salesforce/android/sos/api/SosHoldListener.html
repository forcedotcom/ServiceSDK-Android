SosHoldListener

com.salesforce.android.sos.api.SosHoldListener

Class Overview

Listener interface for receiving changes to the session hold state.

Example Usage

 
   SosHoldListener holdListener = new SosHoldListener() {
     @Override
     public void onHoldStateChanged (SosHoldState holdState, SosHoldState previousHoldState) {
     // The session's HOLD state has changed
     }
   };

   // Register the listener with SOS
   Sos.addHoldListener(holdListener);

   // Unregister the listener and stop receiving HOLD updates
   Sos.removeHoldListener(holdListener);

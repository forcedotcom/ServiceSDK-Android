public interface

SosNetworkTestListener

com.salesforce.android.sos.api.SosNetworkTestListener

Class Overview

Interface for listening to the status of the network test conducted at the beginning of a session.

You can use this interface to be alerted to the status of the network test that SOS performs by default prior to starting a session (the network test may be disabled via networkTestEnabled(boolean)). This may be useful to, for example, display a custom UI during the network test.

Usage Example

 

   SosNetworkTestListener myListener = new SosNetworkTestListener {
    @Override
    public void onNetworkTestStarted () {
      // The network test has been started. You might use this callback to show a custom UI.
    }

    @Override
    public void onNetworkTestProgress (int progress, int total) {
      // The network test has completed {progress} steps out of {total} that will be executed.
      // The test will be completed after {progress} == {total}.
    }

    @Override
    public void onNetworkTestEnded (boolean completed, boolean passed) {
      // The network test has ended.
      // A false value for {completed} indicates that the test was aborted due to a network error.
      // If {passed} is false then the session will soon end with an InsufficientNetwork reason.
    }

   };

   Sos.addNetworkTestListener(myListener);

   // ... sometime later
   Sos.removeNetworkTestListener(myListener);

public abstract void onNetworkTestEnded (boolean completed, boolean passed)

The network test has ended. This may be called due to an error, or the test completing fully with either a passing or failing grade.

Parameters
completed Whether the test has completed or not. A false value here indicates the test was aborted due to a network error.
passed Whether the test was passed or not. This will always be false if the completed parameter is also false.

public abstract void onNetworkTestProgress (int progress, int total)

The network test has completed at least one step.

Parameters
progress The completed step number.
total The total number of steps in the test.

public abstract void onNetworkTestStarted ()

The network test has begun. This will only be called once per session, and the listener must be added prior to the beginning of the network test to receive this call.

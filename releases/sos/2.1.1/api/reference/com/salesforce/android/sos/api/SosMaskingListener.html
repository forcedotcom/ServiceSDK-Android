public interface

SosMaskingListener

com.salesforce.android.sos.api.SosMaskingListener

Class Overview

Interface for listening for changes to the state of the masked fields during an SOS session.

You can use this interface to be alerted to masked fields being hidden and exposed. Masked fields are initially hidden from the user when the session starts, but may become exposed and re-hidden as the session progresses. Screen capturing always pauses while masked fields are exposed.

Usage Example

 

   SosMaskingListener myListener = new SosMaskingListener () {
     @Override
     public void onMaskedFieldsExposed () {
       // Called whenever one or more masked fields are exposed during the session.
     }

     @Override
     public void onMaskedFieldsHidden () {
       // Called whenever the masked fields are re-hidden during the session.
     }
   };

   Sos.addMaskingListener(myListener);

   // Sometime later...
   Sos.removeMaskingListener(myListener);

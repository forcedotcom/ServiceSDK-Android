Class Overview

Base class for any requests that are made that potentially hit the online Knowledge Base. Specifies the high-level cache handling for the request.

public boolean cacheResults ()

Used to determine if the results from this request to the online Knowledge Base should be cached for later offline access.

Returns
  • True if the results should be cached in a local database.

public boolean returnCachedResults ()

Determines whether results from the offline database should be returned for this request. If false, only results from the online knowledge base will be returned.

Returns
  • True if results from the offline database should be returned if appropriate (e.g. if the device is offline).

public boolean returnIntermediateResults ()

Determines whether both the offline and online results should be returned for a single request.

Returns

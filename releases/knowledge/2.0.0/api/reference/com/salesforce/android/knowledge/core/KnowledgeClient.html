public abstract void addUserLogoutListener (LogoutListener client)

Add a callback to be called when the Kb user is logged out.

The Kb user is the account set using the withUserAccount(UserAccount) method. On this callback the application should call cleanup() and stop using the KnowledgeClient.

If the user specified by withUserAccount(UserAccount) is not valid at kb launch this callback will not be triggered.

public abstract void cleanup ()

Cleans up the client to prevent memory leaks. This should be called whenever a core client is no longer needed. If the KnowledgeClient is not configured using the withUserAccount(UserAccount) method or if the Knowledge UI is used this call is not required.
